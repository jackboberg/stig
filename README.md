# stig

A SQLite migration and schema CLI.

> **Status:** Pre-alpha. No functionality implemented yet — the design is the source of truth.

See [`SPEC.md`](./SPEC.md) for the full design document: CLI surface, file formats,
architecture, and the issue backlog driving v0.1.0.

## Development

Git hooks are defined in [`hk.pkl`](./hk.pkl) using [hk](https://hk.jdx.dev). To install them locally:

```sh
hk install
```

See the hk docs for [installation options](https://hk.jdx.dev/getting_started.html) (`mise use hk`, `brew install hk`, `cargo install hk`).
